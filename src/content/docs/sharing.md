# Sharing Properties (Access Management)

Only property owners can share properties with others.

## Grant Access to Another User

1. Open the property you want to share
2. Scroll to the **"Shared Access"** section (visible on the Overview tab)
3. See the current list of people with access
4. Click the blue **"Grant Access"** button (top-right of the card)
5. In the modal that opens:
   - **Enter email:** Type the person's Gmail address
   - **Select access level:**
     - **Viewer:** Can see everything but cannot make changes
     - **Manager:** Can view and edit tenants, units, rent entries, and expenses
6. Click **"Grant Access"**

### What Happens Next

- If the user already has a Popaty account, they see the property immediately
- If they don't have an account yet, access is pending until they sign in with that Gmail
- You see them listed in the "Shared Access" section

## Update Someone's Access Level

1. Find the person in the **"Shared Access"** list
2. Click **"Update Role"**
3. Choose the new role (Viewer or Manager)
4. Click **"Update"**

## Revoke Access

1. Find the person in the **"Shared Access"** list
2. Click the **"Revoke"** button (or Ã— icon)
3. Confirm the action

They immediately lose access to the property.
