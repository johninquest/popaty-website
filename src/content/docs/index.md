# Popaty User Manual

Welcome to Popaty! This documentation will help you get started with managing your rental properties.

## What is Popaty?

Popaty is a property management application that helps you organize and track:

- Multiple properties and units
- Tenant information and contacts
- Rent payments and history
- Property expenses
- Shared access with team members

## Quick Start

New to Popaty? Start here:

1. [Getting Started](/docs/getting-started) - Sign in and understand your dashboard
2. [Properties](/docs/properties) - Create and manage your properties
3. [Sharing](/docs/sharing) - Grant access to others
4. [Units](/docs/units) - Organize multi-unit properties

Popaty uses Google authentication exclusively. You need a Gmail account to use the application.
